# Installation

## Stable release

To install mcp_email_triage, run this command in your terminal:

```sh
uv add mcp_email_triage
```

Or if you prefer to use `pip`:

```sh
pip install mcp_email_triage
```

## From source

The source files for mcp_email_triage can be downloaded from the [Github repo](https://github.com/marvoosh/mcp_email_triage).

You can either clone the public repository:

```sh
git clone git://github.com/marvoosh/mcp_email_triage
```

Or download the [tarball](https://github.com/marvoosh/mcp_email_triage/tarball/master):

```sh
curl -OJL https://github.com/marvoosh/mcp_email_triage/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd mcp_email_triage
uv pip install .
```
