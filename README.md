# mcp_email_triage

![PyPI version](https://img.shields.io/pypi/v/mcp_email_triage.svg)
[![Documentation Status](https://readthedocs.org/projects/mcp_email_triage/badge/?version=latest)](https://mcp_email_triage.readthedocs.io/en/latest/?version=latest)

MCP server that connects to email providers (Gmail, Outlook) and provides intelligent email management through categorization, summarization, priority detection, and action item extraction.
